# Hazard review checklist

- [ ] `:asil:` is set to one of QM / A / B / C / D
- [ ] Severity, exposure, and controllability are stated (S/E/C rationale)
- [ ] `:scenario:` describes the operational situation that exposes the hazard
- [ ] Hazard is mitigated by at least one `safety_goal`
